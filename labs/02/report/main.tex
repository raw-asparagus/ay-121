\subsection{UTC/PST/LST/JD Conversion Relations}
\label{sec:time_conversion_checks}

The timestamp pipeline is
\begin{align}
  \mathrm{PST} &= \mathrm{UTC} - 8~\mathrm{hr},\\
  \mathrm{PDT} &= \mathrm{UTC} - 7~\mathrm{hr},\\
  \mathrm{JD}  &= \mathrm{julian\_date}(t_{\mathrm{unix}}),\\
  \mathrm{LST} &= \mathrm{lst}(\mathrm{JD},\lambda_{\mathrm{obs}}),
\end{align}
with \texttt{ugradio.timing} as the primary implementation.

\subsection{Rotation-Matrix Pointing Formulation}
\label{sec:pointing_rotation_checks}

For a target with equatorial coordinates $(\alpha,\delta)$,
hour angle $H=\mathrm{LST}-\alpha$, and observatory latitude $\phi$,
we use direction cosines
\begin{equation}
\hat{n}_{\mathrm{eq}}=
\begin{bmatrix}
\cos\delta\cos\alpha\\
\cos\delta\sin\alpha\\
\sin\delta
\end{bmatrix},
\end{equation}
and a rotation-matrix mapping to local horizontal coordinates
\begin{equation}
\hat{n}_{\mathrm{hor}} = R_y\!\left(\phi-\frac{\pi}{2}\right)R_z(H)\hat{n}_{\mathrm{eq}}.
\end{equation}
The commanded horizontal coordinates are then
\begin{equation}
\mathrm{alt}=\arcsin(\hat{n}_{\mathrm{hor},z}), \qquad
\mathrm{az}=\mathrm{atan2}(\hat{n}_{\mathrm{hor},y},\hat{n}_{\mathrm{hor},x}).
\end{equation}

For the required Week-2 target $(l,b)=(120^\circ,0^\circ)$, the report
provides the Galactic-to-equatorial conversion and resulting alt/az track.

The matrix pipeline is packaged in \texttt{ugradiolab.pointing}
(\texttt{galactic\_to\_equatorial\_matrix}, \texttt{equatorial\_to\_altaz\_matrix})
and the operational acquisition path is integrated via \texttt{compute\_pointing(...)}.

\subsubsection{Power-Meter Voltage Cross-Check}
\label{sec:powermeter_voltage}

For a matched $50\Omega$ system, power-meter readings can be converted to
an equivalent sinusoidal voltage scale:
\begin{align}
P_{\mathrm{W}} &= 10^{-3}\,10^{P_{\mathrm{dBm}}/10},\\
V_{\mathrm{rms}} &= \sqrt{50\,P_{\mathrm{W}}},\\
V_{\mathrm{pp}} &= 2\sqrt{2}\,V_{\mathrm{rms}}.
\end{align}
This conversion is used only as a consistency cross-check on the SDR voltage
calibration, not as the sole calibration path, because it assumes:
\begin{itemize}
\item valid $50\Omega$ source/load matching at the measurement frequency,
\item negligible reflection/VSWR error at the calibration point,
\item waveform shape known well enough to map $V_{\mathrm{rms}}$ to $V_{\mathrm{pp}}$.
\end{itemize}

\subsubsection{Datasheet-Driven Chain Gains}
\label{sec:datasheet_chain}

Receiver-chain gains and losses are extracted from component datasheets
rather than inferred from nominal block-diagram values alone.
The current block diagram gives nominal stage targets
(e.g., preamp stages near 26\,dB and IF amplification near 20\,dB),
but final calibration tables in this report use exact part-number datasheet
entries once component IDs are confirmed:
the currently stored NPZ metadata and notebooks do not yet include rooftop/lab
amplifier and filter part numbers, so these IDs must be logged explicitly at
the bench for a complete datasheet table.
\begin{equation}
\mathrm{chain\ gain}(\nu) = \sum_i G_i^{\mathrm{datasheet}}(\nu) - \sum_j L_j^{\mathrm{datasheet}}(\nu).
\end{equation}
Final insertion item: \tbd{CHAIN\_DATASHEET\_TABLE\_WITH\_PART\_NUMBERS}.

\section{Temperature Calibration Experiments}
\label{sec:temperature_cal}

This section converts calibrated power observables into antenna/system temperature units.

\subsection{Hot/Cold Calibration and Y-Factor Framework}
\label{sec:yfactor_main}

Using hot-load and cold-reference spectra:
\begin{equation}
  Y = \frac{P_{\mathrm{hot}}}{P_{\mathrm{cold}}},
\end{equation}
\begin{equation}
  T_{\mathrm{rx}} = \frac{T_{\mathrm{hot}} - YT_{\mathrm{cold}}}{Y-1}.
\end{equation}

Provisional per-LO outputs to be filled after revision:
\begin{itemize}
\item $Y_{1420}=\tbd{Y1420}$, $T_{\mathrm{rx},1420}=\tbd{TRX\_1420\_K}$,
\item $Y_{1421}=\tbd{Y1421}$, $T_{\mathrm{rx},1421}=\tbd{TRX\_1421\_K}$,
\item systematic terms: \tbd{TEMP\_CAL\_SYSTEMATICS\_SUMMARY}.
\end{itemize}

\subsection{Temperature Scale Assumptions and Aperture-Efficiency Link}
\label{sec:temp_scale_assumptions}

The conversion from antenna temperature to inferred brightness temperature is represented as
\begin{equation}
  T_B = \frac{T_A}{\eta_{\mathrm{ap}}\,\eta_{\mathrm{bf}}},
\end{equation}
with provisional efficiency placeholders
$\eta_{\mathrm{ap}}=\tbd{APERTURE\_EFFICIENCY}$ and
$\eta_{\mathrm{bf}}=\tbd{BEAM\_FILL\_FACTOR}$.

From currently available data, this report also estimates an effective
combined efficiency
\begin{equation}
\eta_{\mathrm{eff}} \equiv \eta_{\mathrm{ap}}\eta_{\mathrm{bf}}
= \frac{T_{A,\mathrm{line,obs}}}{T_{B,\mathrm{line,ref}}}.
\end{equation}
Using \texttt{labs/02/scripts/estimate\_eta\_eff.py} and the output
\texttt{labs/02/report/eta\_eff\_estimate.csv}, the provisional estimate is
\begin{equation}
\eta_{\mathrm{eff}} = 0.64 \pm 0.28\ (1\sigma),
\qquad
\eta_{\mathrm{eff},95\%} \in [0.08,\,1.19].
\end{equation}
This estimate is intentionally conservative and includes broad systematics
(receiver-scale uncertainty, offline-baseline noise, and assumed reference
brightness uncertainty for the $(l,b)=(120^\circ,0^\circ)$ field).

\subsection{Week-1 First Measurement: SGP Cold-Sky Substitution}
\label{sec:week1_sgp}

The handout asks for a first system-check line measurement at zenith.
In this report, that check is intentionally replaced by a South Galactic Pole
(SGP) cold-sky measurement as a documented deviation from the literal zenith
instruction.
The reason is calibration quality: for Berkeley, zenith can pass through low-$|b|$
Galactic emission windows during the sidereal day, while SGP is persistently
low in HI column and therefore a cleaner $T_{\mathrm{cold}}$ reference.

The SGP is the direction of minimum HI column density
($N_{\rm HI}\lesssim1.7\times10^{20}$\,cm$^{-2}$, HI4PI survey), guaranteeing a
nearly HI-free baseline spectrum regardless of LST and providing an unambiguous
system-noise confirmation prior to the Week-2 science field.

Figure~\ref{fig:sgp_ta_spectrum} shows the calibrated antenna temperature $T_A(\nu)$
for the SGP LO1420 combined observation
(March 5, 2026, 12:35--12:50 PST; UTC 20:35--20:50;
$\mathrm{JD}\approx2461105.3631$; $\mathrm{LST}\approx23.47\,\mathrm{h}$).
The calibration applies the Y-factor result ($T_{\rm rx,1420}=215.0\pm10.4\,$K) and
the frequency-dependent cold-reference profile from \texttt{calibration\_results.npz}:
\begin{equation}
  T_A(\nu) = P_{\rm obs}(\nu)\,\frac{T_{\rm cold}+T_{\rm rx}}{P_{\rm cold,ref}(\nu)} - T_{\rm rx}.
\end{equation}
The LSR velocity correction for the SGP direction ($l=0^\circ$, $b=-90^\circ$)
at the time of observation is
$\Delta v_{\rm LSR}\approx-7.02$\,km\,s$^{-1}$,
computed via \texttt{ugradio.doppler.get\_projected\_velocity}.
No 21-cm HI emission line is detected anywhere in the spectrum, consistent with the
near-zero column density expected at $b=-90^\circ$.
The continuum level near $T_A\approx T_{\rm cold}=30\,$K confirms that the system
is operating at the expected noise floor, validating the calibration chain before
the main science observations.

\subsection{Frequency Switching and Line-Shape Extraction}
\label{sec:line_shape_main}

From frequency-switched on/off spectra:
\begin{equation}
  r_{\mathrm{line}}(\nu)=\frac{s_{\mathrm{on}}(\nu)}{s_{\mathrm{off}}(\nu)}.
\end{equation}

Intensity-calibrated form:
\begin{equation}
  T_{\mathrm{line}}(\nu)=r_{\mathrm{line}}(\nu)\,G,
\end{equation}
with $G=\tbd{GAIN\_K\_PER\_NORM\_POWER}$.

Consistent with the lab instructions, stacked spectra are compared using
both channel-wise means and medians before adopting a final estimator:
\begin{itemize}
\item mean stack output: \tbd{LINE\_STACK\_MEAN\_SUMMARY},
\item median stack output: \tbd{LINE\_STACK\_MEDIAN\_SUMMARY},
\item adopted smoothing width in channels: \tbd{LINE\_SMOOTH\_NCHAN}.
\end{itemize}
The smoothing choice is constrained by the requirement not to wash out
HI features narrower than about 1 km s$^{-1}$.
Band-edge placement is also verified against FIR roll-off to avoid aliasing-sensitive
channels; final evidence will be summarized by \tbd{FIR\_EDGE\_PLACEMENT\_SUMMARY}.

\subsection{Velocity Axes and LSR Correction}
\label{sec:velocity_main}

Radio-definition Doppler mapping:
\begin{equation}
  v \approx c\,\frac{\nu_0-\nu}{\nu_0},
\end{equation}
with LSR correction
\begin{equation}
  v_{\mathrm{LSR}}=v_{\mathrm{topo}}+\Delta v_{\mathrm{LSR}},
\end{equation}
where $\Delta v_{\mathrm{LSR}}=\tbd{LSR\_SHIFT\_KMS}$ km s$^{-1}$.

\subsection{Baseline Removal and Gaussian Decomposition}
\label{sec:gauss_main}

Baseline removal uses off-line channels and low-order polynomial (or equivalent) subtraction,
followed by multi-component Gaussian fitting.
The default off-line fitting windows are
$[-1.2,-0.6]$ MHz and $[0.6,1.2]$ MHz relative to line center, matching the
Lab 2 guidance. The main implementation uses \texttt{np.polyfit}
for baseline modeling and \texttt{gaussfit}/\texttt{gaussval}
for component fits, with alternatives treated as sensitivity checks.

Main-text provisional science outputs:
\begin{itemize}
\item number of adopted components: \tbd{N\_GAUSS\_FINAL},
\item component centroids (topocentric): \tbd{VTOPO\_COMPONENTS\_KMS},
\item component centroids (LSR): \tbd{VLSR\_COMPONENTS\_KMS},
\item FWHM values: \tbd{FWHM\_COMPONENTS\_KMS},
\item fit quality: $\chi_r^2=\tbd{CHI2\_RED\_FINAL}$.
\end{itemize}

\section{Detailed Equipment Calibration Models}
\label{sec:app_cable_models}

\subsection{Known-Length Cable Models}
\label{sec:app_known_length_models}

For SDR channels at two LOs:
\begin{equation}
  y_i(L)=B_i-\alpha L, \qquad i\in\{1420,1421\}.
\end{equation}

Reflection-aware extension:
\begin{equation}
  y_i(L)=B_i-\alpha L + A_i\cos(kL+\phi_i).
\end{equation}

These alternatives are retained for sensitivity bracketing and model-risk discussion.

\subsection{Unknown-Length Inversion}
\label{sec:app_unknown_length}

Given fitted intercept and slope,
\begin{equation}
  L_i = \frac{B_i-y_i^{\mathrm{obs}}}{\alpha}.
\end{equation}

The reported unknown length uses a combined estimate across channels and
bootstrap confidence intervals.

\subsection{Termination and Reflectometry Derivation}
\label{sec:app_reflectometry_derivation}

From a measured round-trip delay $\Delta t$ on a cable of length $L$,
\begin{equation}
  v_{\mathrm{coax}} = \frac{2L}{\Delta t}.
\end{equation}

Final entries in this section will include state-by-state waveform evidence,
frequency dependence, and uncertainty propagation.
